# Command-line app

To import a Twitter list in command-line

```
LIST_NAME='Women in Data Science'
USER_NAME='BecomingDataSci'
php ./bin/console devobs:import-publishers-lists --list-restriction="${LIST_NAME}" "${USER_NAME}"
```
